git status — Где я? (Показывает, какие файлы изменены).
---------------
После изменений:

git add .

git commit -m "Что сделал"

git push
------------
Если изменил сразу на GitHub и нужно подтянуть все изменения в VSCode 

git pull
-------------

ОТКАТЫВАЕМ НАЗАД 
-Если ещё не закомитили(точка - все файлы)
Проще говоря, если накосячил в VScode и хочу вернуть к последнему варианту который был загружен на GitHub.

git restore .

ОТКАТЫВАЕМ НАЗАД если уже закомитил.
Команда откатит на один шаг назад.
git reset --hard HEAD~1
-------------------


			ВЕТКИ
Флаг -b означает "создать новую ветку и сразу перейти в нее, а дальше название ветки.

git checkout -b update-design
